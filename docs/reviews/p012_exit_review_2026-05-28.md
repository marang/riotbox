## Exit Decision

P012 is closed for the bounded timing-foundation phase.

This does not mean Riotbox has a production-grade arbitrary-audio BPM/downbeat
detector. It means the Source Timing contract, honest degraded-state behavior,
Source Map projection, observer/audio proof path, and all timing-aware lane
output gates are implemented enough to become a regression baseline for P013.

## Boundaries

P012 remains a bounded contract/output-shape proof:

- short real loops may be useful while still requiring manual confirmation
- ambiguous downbeat rows must stay manual-confirm-only or fallback-backed
- weak/unavailable timing must remain visibly degraded
- P013 may improve lane musical depth, but it must reuse the P012 Source Graph,
  Source Timing summary, observer/audio, and all-lane proof contracts

Deferred to P012+ / later phases:

- production-grade arbitrary-audio beat/downbeat detection
- host-audio monitoring
- a new scheduler or arrangement model
- lane sound redesign unrelated to timing proof
- full arrangement export
